Desciption

SPLIT is an easy cutomizable v-card/portfolio template made entierly 95% with html and css it allows you to introduce yourself with the most important informations to your features clients or employers.
The template is responsive that's allows you to view it in the mobile devices also.
The template come whit a 3 sets of colors : red, orange and green to give you a lot of variety in the design .

All the html ,css, javascript code are well organized and commented to make any change easy to do without any problems.

Theme installation

When you download the main download file you finde several folder  :

  1. HTML folder this is the folder that contain all the files of the template
  2. LICENSES folder this folder contain the licenses of the elements used.
  3. DOCUMENTATION : this folder contain the documentation of the theme ,please read the documentation carefully for further information

 

You are concerned with the HTML folder because it contain the main files of the site.You can upload it to your server and start work with it, by doing the customization that you need, color , layout , content, image ......

Add tab section

The SPLIT vcard use the tab system setup powred by the easytabs jquery plugin.

To add a menu link   follow the steps below :

 

Open index.html and first locate the navigation html code to add the tab link to the menu tab :

<!--Main navigation-->
		<nav id="main-nav" class="main-nav tabbed">

	<ul>
		<li ><a class="tab-about" href="#about"><i class="icon-user"></i>About</a></li>
		<li><a class="tab-portfolio" href="#portfolio"><i class="icon-camera"></i>Portfolio</a></li>
		<li><a class="tab-blog" href="#blog"><i class="icon-chat"></i>Blog</a></li>
		<li><a class="tab-contact" href="#contact"><i class="icon-phone"></i>Contact</a></li>
	</ul>

</nav>
	<!--End main navigation-->

 

To add a tab menu to the navigation simply add an li elements to the list with the proper href="#name of the tab"

 

		<li><a class="tab-contact" href="#contact"><i class="icon-phone"></i>Contact</a></li>

 

To add the content of the new tab open index.html and add a section of the new tab to the html code, usally the code will look like below :

 

    <!--Example Tab-->
                            <section id="example" class="bgWhite ofsInBottom">

                                <!--Example -->
                                <div class="example">

                                  new Content here
                        
                                        </div>
                                        <!--End example-->

                                    </section>
        <!--End example tab-->

 

 

Fontello Icons

SPLIT V-card uses fonttello icons , the template come with a set of icons that you can use as you want, and change the default icons to fit your needs.

You can find the list of the icons in the fontello.css in the css  folder then you can choose the icon that you want and change it in the html code

Map setup

SPLIT come with a map location section to locate your adress in google map. To setup your coordinate open index.html file then scroll down until you find the map script setup. You are concerned with the lat = -33.867487, lng = 151.20699 values change them to your coordinate location in the map.

Contact form setup

SPLIT come whith a good to go ajax contact form ready to use, you have to replace the adress of the recipient in the send.php file by your adresse.

To do that open the send.php file located in the main download file in any editor of your choice and go to the line 34  and change the adress by your adresse.
